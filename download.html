Cloud Server: Ubuntu 16.04 (recommended)

{% highlight bash %} # Requires Ubuntu 16.04 (Xenial) sudo add-apt-repository -y ppa:opencpu/opencpu-2.0 sudo apt-get update sudo apt-get upgrade # Installs OpenCPU server sudo apt-get install -y opencpu-server # Done! Open http://yourhost/ocpu in your browser # Optional: installs rstudio in http://yourhost/rstudio sudo apt-get install -y rstudio-server {% endhighlight %}

Local Single-User Server

{% highlight r %} # Install OpenCPU install.packages("opencpu") # Run Apps directly from Github library(opencpu) ocpu_start_app("rwebapps/nabel") ocpu_start_app("rwebapps/markdownapp") ocpu_start_app("rwebapps/stockapp") # Install / remove apps installed_apps() remove_apps("rwebapps/stockapp") {% endhighlight %}

Docker (new!)

Images based on various platforms are published on dockerhub. The opencpu/rstudio is great for development: it runs both opencpu-server and rstudio-server:

{% highlight r %} # Run server as executable docker run --name mybox -t -p 80:80 opencpu/rstudio # OR: if port 80 is taken use port 8004 docker run --name mybox -t -p 8004:8004 opencpu/rstudio # OR: to run server in background docker run --name mybox -t -d -p 80:80 opencpu/rstudio {% endhighlight %}

Now simply open http://localhost/ocpu/ and http://localhost/rstudio/ in your browser! Login via rstudio or shell with user: opencpu (passwd: opencpu).

To start a root shell to connect to a running container:

{% highlight r %} # Replace 'mybox' with the container name or id docker exec -i -t mybox /bin/bash {% endhighlight %}

For docker deployments, use the opencpu/base image. See the docker readme for more details.
