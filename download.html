Ubuntu 14.04 (recommended)

{% highlight bash %} #requires ubuntu 14.04 (Trusty) sudo add-apt-repository -y ppa:opencpu/opencpu-1.5 sudo apt-get update sudo apt-get install -y opencpu # optional: sudo apt-get install -y rstudio-server {% endhighlight %}

Fedora

Go here to add the opencpu repository. Also have look at the readme for rpm packages.

{% highlight r %} sudo yum install opencpu {% endhighlight %}

Debian / CentOS / Suse etc

See the readme on building deb or rpm packages from source.

Docker

Docker containers based on various platforms are published via dockerhub:opencpu. Use opencpu/base for opencpu or opencpu/rstudio for opencpu + rstudio server. See this readme for more details. {% highlight r %} # Run as executable docker run -t -p 80:80 -p 8004:8004 opencpu/rstudio # Run in background docker run -t -d -p 80:80 -p 8004:8004 opencpu/rstudio {% endhighlight %}

Local Single-User Server

{% highlight r %} install.packages("opencpu") library(opencpu) {% endhighlight %}

Installing OpenCPU Apps

{% highlight r %} #install the "gitstats" or "stocks" app from github library(devtools) install_github("opencpu/gitstats") install_github("opencpu/stocks") #open in single-user server library(opencpu) opencpu$browse("/library/gitstats/www") {% endhighlight %}
