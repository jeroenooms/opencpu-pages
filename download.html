Ubuntu 14.04 Cloud Server (recommended)

{% highlight bash %} #requires ubuntu 14.04 (Trusty) sudo add-apt-repository ppa:opencpu/opencpu-1.4 sudo apt-get update sudo apt-get install opencpu sudo apt-get install rstudio-server #optional {% endhighlight %}

Fedora, SUSE Cloud Server

Go here to add the opencpu repository. Also have look at the readme for rpm packages.

{% highlight r %} sudo yum install opencpu {% endhighlight %}

Debian / CentOS / etc

See the readme on building deb or rpm packages from source.

Docker

Docker containers with various platforms are published via dockerhub. If you are not sure which one to pick, use the opencpu/stable. {% highlight r %} # Run as executable docker run -t -p 80:80 -p 443:443 opencpu/stable # Run in background docker run -t -d -p 80:80 -p 443:443 opencpu/stable # With shell docker run -t -i -p 80:80 -p 443:443 opencpu/stable sh -c 'service opencpu restart && /bin/bash' {% endhighlight %}

Local Single-User Server

{% highlight r %} install.packages("opencpu") library(opencpu) {% endhighlight %}

Installing OpenCPU Apps

{% highlight r %} #install the "gitstats" or "stocks" app from github library(devtools) install_github("opencpu/gitstats") install_github("opencpu/stocks") #open in single-user server library(opencpu) opencpu$browse("/library/gitstats/www") {% endhighlight %}
