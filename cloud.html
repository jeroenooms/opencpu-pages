OpenCPU Cloud Server and ocpu.io

Deploy packages and apps automatically on our cloud server (or on your own!) by configuring the webhook in your Github repository.

The yourname.ocpu.io domain provides a short URL to your personal app library on the cloud server.

URL at ocpu.io Alias (Long URL) Repository
https://<github-user>.ocpu.io/ https://cloud.opencpu.org/ocpu/apps/<github-user>/ User's Github
https://cran.ocpu.io/ https://cloud.opencpu.org/ocpu/library/ CRAN Packages
https://tmp.ocpu.io/ https://cloud.opencpu.org/ocpu/tmp/ Sessions
